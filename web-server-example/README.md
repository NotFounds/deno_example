# Web Server Example
A web server example. Using deno and oak.

## Requirements
- deno v1.9.0
- oak v7.3.0
- velociraptor v1.0.0-beta.18
  Task Runner

## Usage
```sh
$ # Run deno fmt and deno lint
$ vr check

$ # Run test
$ vr test

$ # Serve a web server on localhost:8000 and watch files for development
$ vr serve-dev

$ # Serve a web server on localhost:8000
$ vr serve
```
